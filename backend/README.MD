## About

O backend usado para fazer criar usuário e se logar. Usado no mobile Flutter.

## Run Project

Clone Project

```git
git clone https://github.com/iury0393/authentication_node_flutter
cd backend
yarn install
```
**Modifique o arquivo ormconfig com seus dados do banco***
