## Development setup

Clone Project

```git
git clone https://github.com/iury0393/authentication_node_flutter
cd authentication_node_flutter
```

API

```ssh
cd backend
yarn
yarn typeorm migrattion:run
yarn dev:server
```

Mobile

```ssh
cd mobile
pub get
start the application
```
